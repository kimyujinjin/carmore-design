# ComparisonTable

> 렌터카 또는 숙소 옵션을 나란히 비교하는 테이블 컴포넌트

---

## 1. 설명 (Description)

ComparisonTable은 사용자가 여러 상품을 **항목별로 나란히 비교**할 수 있는 컴포넌트입니다. Carmore에서는 렌터카 모델 비교(가격, 좌석 수, 연료, 트렁크 크기, 보험 조건)와 숙소 옵션 비교(가격, 평점, 편의시설, 거리)에 사용됩니다.

ComparisonTable의 variant는 **뷰포트 환경**에 따라 선택합니다:
- **Side-by-side**: 데스크톱에서 2~3개 컬럼을 한 화면에 표시
- **Stacked**: 모바일에서 세로로 쌓아 보여주는 컬럼 뷰

## 2. 해부도 (Anatomy)

```
┌────────┬────────────┬────────────┬────────────┐
│        │ ①[Thumb]   │ ①[Thumb]   │ ①[Thumb]   │  ← Header Row
│        │ ②[Name]    │ ②[Name]    │ ②[Name]    │
│        │    [✕]     │    [✕]     │    [✕]     │  ③ Remove Button
├────────┼────────────┼────────────┼────────────┤
│ ④Label │ ⑤[Value]   │ ⑤[Value]   │ ⑤[Value★]  │  ⑥ Highlight Cell
├────────┼────────────┼────────────┼────────────┤
│ ④Label │ ⑤[Value]   │ ⑤[Value★]  │ ⑤[Value]   │
├────────┼────────────┼────────────┼────────────┤
│ ④Label │ ⑤[Value]   │ ⑤[Value]   │ ⑤[Value]   │
└────────┴────────────┴────────────┴────────────┘
   ⑦ Sticky Column (mobile)
```

| # | Part | 필수 여부 | 설명 |
|---|------|-----------|------|
| ① | Product Thumbnail | 선택 | 상품 이미지 썸네일 |
| ② | Product Name | 필수 | 상품명 (차량 모델명 또는 숙소명) |
| ③ | Remove Button | 선택 | 컬럼 제거 버튼 (✕) |
| ④ | Feature Label | 필수 | 비교 항목명 (첫 번째 컬럼, sticky) |
| ⑤ | Feature Value | 필수 | 각 상품의 항목 값 |
| ⑥ | Highlight Cell | 선택 | 최적 값 강조 표시 |
| ⑦ | Sticky Column | 조건부 | 모바일에서 첫 번째 컬럼 고정 |

## 3. 옵션 (Options / Props)

| Property | Type | Values | Default | 설명 |
|----------|------|--------|---------|------|
| `variant` | enum | `"side-by-side"` \| `"stacked"` | `"side-by-side"` | 레이아웃 방식 |
| `columns` | array | max 4 items | — | 비교 대상 상품 데이터 배열 |
| `rows` | array | — | — | 비교 항목(feature) 정의 배열 |
| `highlightBest` | boolean | `true` \| `false` | `false` | 최적 값 자동 강조 |
| `stickyHeader` | boolean | `true` \| `false` | `true` | 스크롤 시 헤더 행 고정 |

---

## 4. 인터랙션 (Interactions)

| 입력 | 동작 |
|------|------|
| **Horizontal Scroll** (모바일) | 첫 번째 컬럼(feature label) 고정, 나머지 컬럼 좌우 스크롤 |
| **Tap Column Header** | 해당 상품 상세 페이지로 이동 |
| **Tap Remove (✕)** | 비교 목록에서 해당 상품 제거 |
| **Vertical Scroll** | stickyHeader=true 시 헤더 행 상단 고정 |
| **Hover Row** (Web) | 행 하이라이트로 가독성 향상 |
| **Keyboard Navigation** | 셀 간 화살표 키 이동 |

## 5. 가이드라인 (Do & Don't)

### DO

- ✅ **비교 항목은 최대 4개까지만 허용합니다**
  ```
  [소나타] vs [K5] vs [말리부]    ← 3개 비교 (적정)
  ```

- ✅ **차이가 나는 항목을 강조 표시합니다** — `highlightBest`로 최적 값 시각화
  ```
  가격:  ₩45,000  [₩38,000★]  ₩52,000    ← 최저가 강조
  ```

- ✅ **동일 카테고리의 상품만 비교합니다** — 렌터카끼리, 숙소끼리

- ✅ **모바일에서는 첫 번째 컬럼을 sticky로 고정하여 항목명 가시성을 유지합니다**

- ✅ **가장 중요한 비교 항목(가격, 평점)을 상단에 배치합니다**

### DON'T

- ❌ **4개를 초과하는 상품을 비교하지 않습니다** — 가독성 저하
  ```
  ❌ 5개 이상 컬럼 → 스크롤이 과도해져 비교 불가
  ```

- ❌ **서로 다른 카테고리의 상품을 비교하지 않습니다**
  ```
  ❌ [소나타 렌터카] vs [해운대 호텔]    ← 비교 불가
  ```

- ❌ **비교 항목이 2개 미만일 때 ComparisonTable을 사용하지 않습니다** — Card 사용 권장

- ❌ **모든 값이 동일한 항목을 비교 행에 포함하지 않습니다** — 차이가 있는 항목만 표시

## 6. 플랫폼 차이 (Platform Differences)

| 항목 | Web | iOS | Android |
|------|-----|-----|---------|
| 레이아웃 | side-by-side (full table) | 수평 스크롤 + sticky column | 수평 스크롤 + sticky column |
| 컬럼 수 | 최대 4개 표시 | 1~2개 보이고 스크롤 | 1~2개 보이고 스크롤 |
| Hover 행 하이라이트 | 지원 | 미지원 | 미지원 |
| 헤더 고정 | sticky (CSS) | sticky (UIKit/SwiftUI) | sticky (Compose) |
| 스크롤 인디케이터 | 스크롤바 (커스텀) | 네이티브 관성 스크롤 | 네이티브 관성 스크롤 |
| 상품 제거 | ✕ 버튼 hover 표시 | ✕ 버튼 항상 표시 | ✕ 버튼 항상 표시 |

## 7. 접근성 (Accessibility)

| 속성 | 값 |
|------|-----|
| **Role** | `table` |
| **Header Scope** | `scope="col"` (상품명 헤더), `scope="row"` (항목명) |
| **aria-label** | 테이블에 "렌터카 비교" 또는 "숙소 비교" 등 설명 |
| **aria-label (highlight)** | 강조 셀에 "최저가", "최고 평점" 등 추가 레이블 |
| **aria-sort** | 정렬 가능한 행에 적용 |
| **키보드** | 화살표 키로 셀 간 이동, `Enter`로 헤더 탭 액션 |
| **스크린 리더** | "[상품명], [항목명], [값]" 순서로 읽기. 강조 셀은 "최적, [값]" |
| **축소 모션** | `prefers-reduced-motion` 시 스크롤 애니메이션 비활성 |

## 8. Carmore 사용 예시

### 렌터카 비교

```
┌──────────┬─────────────┬─────────────┬─────────────┐

├──────────┼─────────────┼─────────────┼─────────────┤
│ 일일 요금  │  ₩55,000    │ [₩48,000★]  │  ₩62,000    │
│ 좌석 수   │  5명         │  5명         │  5명         │
│ 연료      │  가솔린      │  하이브리드   │  가솔린       │
│ 트렁크    │  510L       │ [530L★]     │  500L        │
│ 보험      │  기본        │  완전 포함    │  기본         │
└──────────┴─────────────┴─────────────┴─────────────┘
```

```
│ 평점      │  4.2         │ [4.7★]       │

│ 거리      │ [도심 0.5km★] │  도심 3.2km   │
└──────────┴──────────────┴──────────────┘
```
