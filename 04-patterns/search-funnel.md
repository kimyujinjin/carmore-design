# Search Funnel (검색 퍼널)

> 사용자가 렌터카 또는 숙박을 탐색하는 다단계 검색 플로우

---

## 개요

Search Funnel은 Carmore의 **핵심 사용자 여정**입니다. 사용자가 목적지를 입력하고, 날짜를 선택하고, 필터를 적용하여 결과를 확인하는 일련의 과정을 체계적으로 안내합니다.

---

## 퍼널 단계

```
Step 1          Step 2          Step 3          Step 4          Step 5
위치 입력   →   날짜 선택   →   옵션 설정   →   결과 탐색   →   상세 진입
                                (선택)
```

---

## Step 1: 위치 입력

### 사용 컴포넌트

| 컴포넌트 | Variant / 설정 |
|----------|---------------|
| **TextField** | `variant="search"`, `size="lg"`, `placeholder="어디로 가시나요?"` |

### 동작 흐름

```
1. 사용자가 검색 필드를 탭/클릭
2. 키보드가 올라오며 포커스 상태 활성
3. 입력하면 실시간 자동완성 드롭다운 표시
   - 최근 검색 (아이콘: ic_action_search)
   - 인기 도시 (아이콘: ic_general_location)
   - 공항/역 (아이콘: 해당 카테고리 아이콘)
4. 항목 선택 시 다음 단계로 자동 이동
```

### 레이아웃

```
Mobile:
┌─────────────────────────────────┐
│  ← 뒤로    렌터카 검색           │  ← Navigation Bar
├─────────────────────────────────┤
│  🔍 [어디로 가시나요?         ] │  ← TextField(search, lg)
├─────────────────────────────────┤
│  최근 검색                       │
│  ├─ 🔍 서울 강남                │
│  ├─ 🔍 제주 공항                │
│  인기 도시                       │
│  ├─ 📍 서울                     │
│  ├─ 📍 부산                     │
│  └─ 📍 제주                     │
└─────────────────────────────────┘

Desktop:
┌──────────────────────────────────────────────────┐
│  🔍 [위치]  │  📅 [날짜]  │  👤 [인원]  │ [검색] │  ← 가로 배치
└──────────────────────────────────────────────────┘
  ↕ 드롭다운 표시
```

---

## Step 2: 날짜 선택

### 사용 컴포넌트

| 컴포넌트 | Variant / 설정 |
|----------|---------------|
| **BottomSheet** (Mobile) | `variant="modal"`, 날짜 피커 내장 |
| **TextField** (Desktop) | 클릭 시 캘린더 팝오버 |

### 동작 흐름

```
1. 위치 선택 후 자동으로 날짜 선택 진입
2. 캘린더에서 시작일 탭 → 종료일 탭 (범위 선택)
3. 선택된 날짜 범위 하이라이트 표시
   - 시작일: $color.bg.brand (원형)
   - 범위: $color.bg.brand-weak (연한 배경)
   - 종료일: $color.bg.brand (원형)
4. "적용" 버튼으로 확정
```

### 유효성 검사

| 규칙 | 에러 메시지 |
|------|------------|
| 시작일이 오늘 이전 | "오늘 이후 날짜를 선택해 주세요" |
| 종료일이 시작일 이전 | "반납일은 대여일 이후여야 합니다" |
| 최대 기간 초과 (30일) | "최대 30일까지 예약 가능합니다" |

---

## Step 3: 옵션 설정 (선택)

### 사용 컴포넌트

| 컴포넌트 | 용도 |
|----------|------|
| **FilterChip** (`variant="toggle"`) | 차종, 연료, 편의시설 필터 토글 |
| **FilterChip** (`variant="selection"`) | 가격 범위, 정렬 기준 드롭다운 |
| **BottomSheet** (Mobile) | "더 보기" 필터 상세 패널 |

### 필터 그룹

```
렌터카:
[차종 ▾]  [가격 ▾]  [연료 ▾]  [변속기]  [보험 포함]  [더보기]

숙박:
[숙소유형 ▾]  [가격 ▾]  [편의시설]  [평점 ▾]  [거리순]  [더보기]
```

### 레이아웃

```
┌─────────────────────────────────────────┐
│  ← 가로 스크롤                    →     │
│  [차종▾] [가격▾] [연료▾] [변속기] ...    │  ← FilterChip group
└─────────────────────────────────────────┘
  chip-gap: $dimension.spacing.chip-gap (8px)
  page-margin 안쪽에 배치
  가로 스크롤 시 좌우 fade gradient 표시
```

## Step 4: 결과 탐색

### 사용 컴포넌트

| 컴포넌트 | Variant / 설정 |
|----------|---------------|
| **Card** | `variant="listing"`, 검색 결과 각 항목 |
| **FilterChip** | 상단 고정 필터 영역 |
| **Button** (`variant="tertiary"`) | 정렬 기준 변경 |

### 레이아웃

```
Mobile (세로 리스트):
┌─────────────────────────────────┐
│  "서울 강남" 검색결과 24건        │  ← $typography.subtitle2
│  [추천순 ▾]                      │  ← Button(tertiary, sm)
├─────────────────────────────────┤
│  ┌─────────────────────────┐   │
│  │  [이미지 16:9]           │   │
│  │  현대 아반떼              │   │  ← Card(listing, vertical)
│  │  📍 강남역 3분  ⭐ 4.8    │   │
│  │  ₩45,000/일              │   │
│  └─────────────────────────┘   │
│  ← 8px gap →                   │  ← $dimension.spacing-y.component-gap
│  ┌─────────────────────────┐   │
│  │  [다음 카드]              │   │
│  └─────────────────────────┘   │
└─────────────────────────────────┘

Desktop (그리드):
  2~3열 그리드, Card(listing, vertical)
  column-gap: $dimension.x4 (16px)
  row-gap: $dimension.x6 (24px)
```

### 빈 결과

결과가 없을 경우 → [Empty States 패턴](./empty-states.md) 참조

---

## Step 5: 상세 진입

Card 탭 시 상세 페이지로 이동합니다.

- **모바일**: 화면 전환 (push navigation)
- **데스크톱**: 같은 페이지 또는 새 탭
- **전환 모션**: `$motion.duration.slower` (500ms), `$motion.easing.standard`

---

## 상태 유지 (State Persistence)

| 항목 | 유지 방식 |
|------|-----------|
| 검색 조건 (위치, 날짜, 인원) | URL 쿼리 파라미터 |
| 필터 상태 | URL 쿼리 파라미터 |
| 스크롤 위치 | 뒤로가기 시 복원 |
| 최근 검색 | Local Storage |

```
예시 URL:
/search?location=seoul-gangnam&start=2026-03-15&end=2026-03-18&type=sedan&fuel=gasoline&sort=recommended
```

---

## 데스크톱 vs 모바일 차이

| 항목 | Mobile | Desktop |
|------|--------|---------|
| 검색 바 | 세로 스택, 각 단계 개별 화면 | 가로 배치, 한 줄 |
| 날짜 피커 | BottomSheet 풀스크린 캘린더 | 인라인 팝오버 |
| 필터 | 가로 스크롤 칩 + BottomSheet 상세 | 사이드바 또는 팝오버 |
| 결과 목록 | 세로 리스트 | 2~3열 그리드 |
| 지도 보기 | 탭 전환 (목록 ↔ 지도) | 좌측 리스트 + 우측 지도 분할 |
